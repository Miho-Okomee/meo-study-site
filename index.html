P1｜初回診断

目的
お客様のGBP（Googleビジネスプロフィール）の現状を数値化し、課題と改善の優先順位を可視化する。

ステップ1｜初回訪問・営業

  • 「無料でGBPの現状を診断します」とアプローチ
  • 持参するもの：名刺、案内チラシ（任意）

🎯 目標業種（競合MEO業者が薄い順）

業種月額目安特徴
結婚相談所 月3〜7万円 ROIが高い。競合MEO業者が薄いブルーオーシャン
動物病院 月3〜5万円 GBP放置率が高く改善余地が大きい
士業（税理士・行政書士） 月5〜8万円 長期継続に発展しやすい。単価高め
整骨院・鍼灸院 月3〜5万円 安定需要。施術系は写真映えしやすい
飲食店 ⚠️ — 競合激戦・ROIが出にくいため避ける

ステップ2｜3分GBP診断シート（15項目）

ファイル：meo_gbp_shindan_sheet.html


セクション配点チェック内容
基本情報の設定 各2点／10点満点 カテゴリ・NAP・属性・営業時間・説明文
写真・動画・投稿 各2点／10点満点 枚数・更新頻度・投稿・動画・Q&A
口コミ状況 各2点／10点満点 件数・スコア・新着ペース・返信率・内容の質

採点基準

評価点数判断基準
問題なし2点十分に設定・運用されている
要改善1点設定はあるが内容が不十分
対応が必要0点未設定または長期放置

総合判定

判定スコア意味
A判定 80%以上（24点以上） 良好。維持・強化フェーズ
B判定 50〜79%（15〜23点） 改善余地あり。提案チャンス
C判定 50%未満（14点以下） 緊急改善案件。受注確度高い
💡 営業のコツ：「やらないコスト」を見せる
「近くのA院は月8件ペースで口コミ獲得、御社は月2件」という競合比較を提示する。商品説明ではなく課題の可視化から入る。

P2｜登録・整備

目的
GBPの基本情報をすべて正確に設定し、スコアを初期段階から引き上げる。

GBP設定チェックリスト

ファイル：gbp-practice/index.html（ローカル）でお客様と画面を見ながら一緒に入力・登録

  • メインカテゴリ＋サブカテゴリ（業態に最適化）
  • 店舗名・住所・電話番号（NAP統一 ※HPや他サイトとも一致させる）
  • 属性タグ（Wi-Fi・駐車場・対応決済・バリアフリーなど）
  • 営業時間（祝日・特別日まで設定）
  • ビジネス説明文（750文字以内、地域名＋業種名＋強みを含む）
  • 写真10枚以上（外観・内観・スタッフ・商品）
  • Q&A自社投稿（5件以上）
📊
完了後に診断シートで再採点する
登録前後のスコア差をお客様に見せる。スコアが上がった数字が「改善の証拠」になり、継続契約の説得材料になる。
🚀

P3｜運用スタート ★最初のアドバイスタイミング

目的
お客様が自走できる状態を作る。登録直後の意欲が高いタイミングに渡すことで定着率が大きく上がる。以下4項目をA4一枚にまとめて渡す。

① 口コミ依頼文のテンプレート

お客様が来店後に送れる依頼文（3パターン）。毎回文章を少し変えて使うこと。

🚫
注意点（必ず守る）
  • 同一IPからの連投はNG（口コミが削除される）
  • テンプレート文の使い回しはNG（毎回少し変える）
  • ガイドライン違反ワード（割引・プレゼント・特典等）はNG
パターンA｜LINE用・カジュアル
本日はご来院いただきありがとうございました😊 もしよろしければ、Googleマップへの口コミを書いていただけると大変嬉しいです。 来店された感想を、ひと言でも書いていただけるだけで助かります🙏 【口コミ投稿ページ】 https://g.page/r/（店舗のショートURL） お時間のあるときにぜひよろしくお願いします！
パターンB｜メール用・丁寧
〇〇様 本日はご来店いただきまして、誠にありがとうございました。 お客様のご来店が、私たちの大きな励みになっております。 もしよろしければ、Googleマップにご感想をお寄せいただけますと幸いです。 いただいた口コミは、サービス改善と、同じようなお悩みをお持ちの方への参考として大切に活用させていただきます。 ▼ 口コミ投稿はこちら https://g.page/r/（店舗のショートURL） お忙しいところ恐れ入りますが、何卒よろしくお願い申し上げます。 〇〇（店舗名）スタッフ一同
パターンC｜短文・汎用（SNS・口頭でも使える）
本日はありがとうございました！ Googleマップに感想を書いていただけると嬉しいです⭐ ↓ https://g.page/r/（店舗のショートURL） 一言でも大丈夫です！よろしくお願いします🙏

② 口コミ返信テンプレート

返信の目標：全件・72時間以内（2026年アルゴリズム重要因子）

ポジティブ口コミ用（感謝＋次回来店誘導）
〇〇様、この度は温かいお言葉をいただきありがとうございます。 スタッフ一同、励みになります。またのご来店をスタッフ一同心よりお待ちしております。 引き続きどうぞよろしくお願いいたします。 〇〇（店舗名）
ネガティブ口コミ用（謝罪＋改善姿勢の表明）
〇〇様、この度はご不満をおかけしてしまい、誠に申し訳ございません。 いただいたご意見はスタッフ全員で共有し、サービス改善に努めてまいります。 もしよろしければ、直接ご意見をお聞かせいただける機会をいただけますと幸いです。 〇〇（店舗名）スタッフ一同

  • 動画はGBPエンゲージメントに直結するため優先度高

自動化完成後の月次フロー
1クライアントあたり月1〜1.5時間に圧縮。10クライアントでも月10〜15時間で回せる。

手動

① レポート自動生成

使うAPIGA4 Data API、Search Console API
処理ツールGAS ＋ Claude API
出力先Googleスライド or パワーポイント
難易度中（API認証が最初の山）
効果月2〜3時間削減
月次レポートの構成 1. KPIサマリー（表示回数・クリック・口コミ件数・返信率）
2. 競合比較（1位・2位と並べて比較）
3. 今月の実施施策と効果
4. 来月の施策計画
5. 担当者コメント

② 口コミ返信 自動草案化

使うAPIGoogle Business Profile API
処理ツールGAS ＋ Claude API
難易度低〜中（一番早く動かせる）
効果1件5〜10分 → 確認のみ30秒
プロンプト設計のコツ ・店舗の「口調・キャラクター」をシステムプロンプトに設定
・口コミの星の数・内容・キーワードを入力
・ポジ・ネガを自動判別して文体を変える
・250文字以内で収める設定を入れる

③ 投稿ネタ選定・自動入稿

使うAPIGoogle Business Profile API
処理ツールGAS ＋ Claude API
難易度中
効果月4投稿分の企画・文章・入稿が自動化
⚠️
「その店ならではの体験・実例」を必ず含ませる。AI生成丸出しの汎用文は評価が下がる。写真・動画とセットで投稿する。
